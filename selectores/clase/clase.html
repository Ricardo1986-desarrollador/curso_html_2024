CSS . selector de clase

Definición y uso

El .classselector selecciona elementos con un atributo de clase específico.

Para seleccionar elementos con una clase específica, escriba un carácter de punto (.), seguido del nombre de la clase.

También puede especificar que una clase solo deba afectar a elementos HTML específicos. Para hacer esto, comience con el nombre del elemento, luego escriba el carácter de punto (.), seguido del nombre de la clase (consulte el Ejemplo 1 a continuación).

Los elementos HTML también pueden hacer referencia a más de una clase (consulte el Ejemplo 2 a continuación).
